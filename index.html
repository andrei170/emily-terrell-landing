FAQ

Common questions, before enrolling.

Do I need to be tech-savvy to use this course?

No. The course is built for agents who know their RE workflow but are new to AI. Module 1 starts with tool selection from scratch. If you can use a CRM and write an email, you can run every module in this course.

How fast will I see results?

Most agents install Module 2 (listing prompts) within the first two days of enrolment and use the prompts on their next active listing. The full implementation sprint takes five days. Most agents are running all five modules before the end of the sprint week.

Will this work for my specific market?

Yes. The prompts and systems are built around RE workflow tasks - listing copy, follow-up, CMA, market reports - that apply regardless of market. The prompt library includes examples across residential, luxury, and commercial-adjacent RE contexts.

What is the time commitment?

Each module is approximately 30-45 minutes of video content plus the implementation time for that module's prompts and templates. The full five-day sprint is designed for agents with a full pipeline - not a block-out-your-calendar commitment.

Is this for buyers' agents, listing agents, or both?

Both. The prompt library covers buyer communication, listing copy, CMA, and market reports. Modules 1-2 weight toward listing agents. Module 3 weights toward agents with large buyer pipelines. Module 4 is relevant to any agent who writes CMAs or market reports.

What if I'm already using some AI tools?

Module 1 includes a diagnostic for agents who are already using AI but not getting consistent results. If you are already running prompts, Module 1 helps you identify the gaps in the current setup before Modules 2-5 build on top of it.
